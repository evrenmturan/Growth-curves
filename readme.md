# Growth-curves
Program to test various growth curves against data.

## growth-curves_inverse.py

The following 3 parameter functions are fitted:

    * Gompertz_3a
    * Monomolecular
    * von Bertalanffy
    * Extreme Value
    * He Legendre
    * Korf3          
    * Logisitic3     
    * MMF (Morgan-Mercer-Flodin3a) 
    * Weibull (3a)  
    * MichaelisMenten
    * Negative Exponential
    * Power (2 forms)
    * Chapman Richards
